# InceptionOfThings

This project should be cloned in a Virtual Machine of your choice.

## p1 & p2
### Start a K3s cluster
```bash
cd p1
vagrant up
```

### Access the K3s cluster
You can either use SSH:
```bash
vagrant ssh cde-sous
```
Or you can use a virtual machine manager.

## p3
In your terminal, run: ```./setup.sh```
